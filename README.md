# Work-Space-World-Sales-Analytics-Dashboard-Power-BI-
The Work Space World project analyzes retail sales data (Superstore-style dataset) to deliver insights on sales, profit, customer segments, and regional performance. The dashboard helps business stakeholders track revenue, monitor profitability, and identify growth opportunities across products, regions, and customer segments.

**Project Overview:**

**Data Cleaning & Preparation**

Handled missing values (e.g., dates, product categories).

Removed duplicates and standardized field names.

Fixed data types for date, numeric, and categorical fields.

**Data Modeling**

Designed fact & dimension tables for Orders, Customers, Products, and Regions.

Built relationships between Orders and Returns tables.

Created a Date table for time intelligence (YoY, MoM).

**Analysis & KPIs**

Sales Metrics → Total Sales, Profit, Profit Margin %, Total Quantity.

Regional Analysis → Sales by Region, Country, State, and City.

Customer Segmentation → Consumer, Corporate, and Home Office performance.

Category & Product Analysis → Top/Bottom Products by Sales & Profit.

Returns Analysis → Return count & Return Rate %.

Time Intelligence → YoY Sales, Profit Growth, Seasonal Trends.

**Visualizations**

KPI Cards → Sales, Profit, Profit Margin %, Returns.

Line Chart → Sales/Profit trend over time.

Bar/Column Charts → Sales by Category, Sub-Category, Region.

Tree Map → Customer Segment contribution.

Map Visualization → Geographical sales distribution.

Table → Top N Products with Sales & Profit.

**Business Value**

This dashboard provides business leaders with:

A 360° view of sales performance across products, customers, and regions.

Insights into profitability drivers (high vs low margin products).

Customer segmentation analysis for targeted marketing.

Regional insights to optimize supply chain & distribution.

Tools to reduce losses from returns and improve efficiency.
